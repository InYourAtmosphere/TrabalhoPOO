# Sistema de Gestão de Locadora de Veículos

**Alunos:** Finéias Antônio, Gabriela Brasileiro, Heitor Marx Vaz, Túlio Vitorette

---

## 1. Definição do Problema e Escopo do Trabalho

### Definição do Problema
A gestão atual lida com o uso de planilhas manuais, o que tem causado problemas significativos de eficiência operacional, como a descentralização das informações, a propensão a erros humanos e o constante retrabalho.

### Escopo do Trabalho
O projeto consiste em desenvolver uma plataforma única capaz de integrar todas as unidades da locadora. O escopo central abrange a garantia da integridade e centralização dos dados referentes a contratos, manutenção e quilometragem. Além disso, o sistema deve fornecer gestão dinâmica de inventário, comunicações automatizadas com os clientes, uso de Inteligência Artificial para previsões financeiras e mecânicas, e suporte à decisão gerencial por meio de dashboards baseados em evidências.

---

## 2. Lista de Requisitos

* O sistema deve eliminar o uso de planilhas manuais e integrar todas as unidades em uma plataforma única.
* O sistema deve centralizar e garantir a integridade das informações sobre quilometragem, manutenção e contratos.
* O sistema deve fornecer uma visão gerencial clara e unificada de toda a frota.
* O sistema deve utilizar Inteligência Artificial para prever a depreciação de veículos e realizar manutenções preditivas.
* O sistema deve realizar comunicações automatizadas com o cliente por meio de mensageiros como o WhatsApp.
* O sistema deve prover visibilidade em tempo real da disponibilidade de veículos para aluguel.
* O sistema deve fornecer dashboards em tempo real para auxiliar a diretoria em decisões estratégicas e substituir suposições.
* O sistema deve permitir instanciar (cadastrar) novos veículos, registrando dados como placa, modelo, ano, chassi e unidade física de destino.
* O sistema deve tornar o veículo visível para locação e monitoramento imediato após a sua instanciação.
* O sistema deve permitir a edição de dados para modificar status de veículos, como "em manutenção" ou "disponível".
* O sistema deve permitir atualizar a quilometragem atual e anexar novos documentos aos veículos cadastrados.
* O sistema deve garantir que os dados reflitam sempre o estado físico real do carro, evitando que fiquem obsoletos.
* O sistema deve permitir a transferência lógica de veículos entre diferentes unidades da locadora.
* O sistema deve garantir um controle de estoque fluido e fornecer a cada gerente regional a visibilidade exata da frota sob sua responsabilidade.
